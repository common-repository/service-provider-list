== Description ==

The ideal plugin for display service provider profile list with services on your website.
<br /><br />
You can add many services and includes service provider profile in it. After add profile, you can search by services and manage profile/services parameters in admin "Front Design & HTML variables" section.

== Installation ==

1. Log into WordPress
1. Select Plugins, then "Add New"
1. Select Upload Plugin
1. Choose the zip file, then Install and activate
